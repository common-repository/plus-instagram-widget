== Installation ==



Thanks for installing or downloading our Instagram Widget



1. Install the wordpress plugin first

You can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder



2. Activate the plugin



3. You will see facebook settings from right side of your wordpress menu.



Plus Instagram Widget Settings:



Instagram Widget Allow You To Display Instagram Image In Your Website,



1. Add Widget Title               (example = Plus Instagram Widget)

2. Add Instagram Name             (example = instagram) 

3. Add Width                      (example = 500)

4. Add Height                     (example = 500)

5. Add Instagram Image Horizontal (example = 3)

6. Add Instagram Image Vertical   (example = 3)
